# 会员双试用购买卡片设计

日期：2026-07-20
范围：Flutter 本地产品规则、RevenueCat Package 适配、个人页会员购买弹窗、l10n、Widget/单元测试与公开合同文档。

## 目标

让符合资格的新用户打开会员购买弹窗后，第一眼看到并默认进入月卡“免费 3 天”；同时让年卡“免费 7 天”在自己的套餐卡片中清晰可见。切换套餐时，主按钮、试用天数、试用后本地化价格、自动续费周期与取消披露必须同步变化。

## 授权与非目标

- 本轮只实现和验证本地代码、测试与公开合同文档。
- 不创建或启用 Google Play 年卡 Offer，不修改 RevenueCat、Worker、D1、Secret、变量或 binding。
- 不发起或取消 Sandbox 订阅，不构建或上传 AAB，不推进任何测试轨道，不 push。
- 不新增 entitlement、Product、Package、Offering、会员状态枚举或后端分支。
- Google Play 结算页仍是资格、价格和交易条款的最终事实。

## 方案比较

### 方案 A：保留 ChoiceChip，只增加年卡试用文字

改动最小，但试用仍是标题下方的小号辅助文字，无法满足“新用户首先注意到月卡免费 3 天”的视觉目标；两档差异也主要依赖用户细读。

### 方案 B：完整宽卡，试用利益点成为卡片主层级（采用）

把每个套餐渲染为整张可点击卡片：套餐名和小型推荐标记在上层，试用利益点使用更高字号和字重，试用后价格作为辅助信息，右侧保留明确选择状态。默认选中的月卡通过主题色表面、语义选中状态和主 CTA 共同形成第一视觉焦点。该方案只替换页面内局部 Widget，不改变购买编排。

### 方案 C：重做为对比表、顶部试用 Hero 和动画切换

表达力更强，但会重复展示套餐信息、增加布局和可访问性风险，也超出本轮两档卡片优化范围。

## 产品规则

试用展示完全来自 RevenueCat 当前 Offering 中对应 Package 的 `defaultOption`，并采用失败关闭策略：

- 月卡只接受 `freePhase.billingPeriod == P3D`。
- 年卡只接受 `freePhase.billingPeriod == P7D`。
- 免费阶段必须是正数天，且 `defaultOption.fullPricePhase` 必须存在并提供非空本地化价格。
- 非 3/7 天、周/月单位、缺 `defaultOption`、缺 `freePhase`、缺完整付费阶段或价格为空时，只按普通套餐展示。
- App 不根据本地历史推断资格，也不把其他时长改写成 3 天或 7 天。

默认选择顺序固定为：

1. 可用的月卡 3 天试用；
2. 可用的年卡 7 天试用；
3. 可用的普通年卡；
4. RevenueCat 返回的第一项可购买套餐。

## 需求矩阵

| 月卡 | 年卡 | 默认项 | 卡片与 CTA 预期 |
|---|---|---|---|
| 3 天试用 | 7 天试用 | 月卡 | 月卡突出“免费 3 天”；年卡显示“免费 7 天”；CTA 为月卡 3 天，切年卡后变为 7 天与年度披露 |
| 3 天试用 | 普通 | 月卡 | 月卡显示 3 天；年卡不出现试用；切年卡后显示普通续费文案 |
| 普通 | 7 天试用 | 年卡 | 年卡显示 7 天并默认选中；切月卡后显示普通续费文案 |
| 普通 | 普通 | 年卡 | 不出现试用承诺；沿用年卡优先 |
| 非 3 天/非 7 天 | 任意 | 按有效项决定 | 不把异常时长展示为试用 |
| 非天单位 | 任意 | 按有效项决定 | 不展示试用 |
| 免费阶段存在但完整付费阶段缺失 | 任意 | 按普通套餐决定 | 不展示试用，不生成不完整转正文案 |
| 仅月卡 | 不存在 | 月卡 | 渲染唯一月卡并购买 monthly Package |
| 不存在 | 仅年卡 | 年卡 | 渲染唯一年卡并购买 annual Package |
| 均不存在 | 均不存在 | 无 | 显示既有加载失败与重试，不允许购买 |

## 卡片信息层级

- 月卡始终排在年卡前，方便新用户先扫描到月卡。
- 试用可用时，卡片主利益点使用“免费 3 天”或“免费 7 天”；价格显示为“试用后 {本地化价格} / 月（年）”。
- 试用不可用时，显示普通本地化价格，不暗示优惠。
- 年卡“推荐”保留为小面积次级标记，不压过默认选中月卡的试用利益点。
- 整卡可点击，最小触控高度不低于 72dp；提供 `Semantics(button: true, selected: ...)`，选择状态不只依赖颜色。
- 浅色和深色均从 `ColorScheme` 取表面、文字和主色；不新增品牌色。

## 动态披露

- 月卡试用：前 3 天免费，之后按商店本地化月价每月自动续费，除非试用结束前取消。
- 年卡试用：前 7 天免费，之后按商店本地化年价每年自动续费，除非试用结束前取消。
- 普通套餐：沿用“通过 Google Play 自动续费，可随时取消”。
- 切换套餐必须在同一帧更新卡片选择、CTA 和披露；购买只返回当前选中的 `PremiumPlanId`。
