# M3 第一版真机测试清单

## APK

`build/app/outputs/flutter-apk/app-debug.apk`

## 必测流程

1. 安装 APK，允许相机权限。
2. 打开「离线回放」，点「开始回放」。
3. 观察：骨架是否显示，计数最终是否为 5，性能面板是否有 FPS/e2eMs。
4. 测「暂停」「继续」「重置」。
5. 打开「实时相机」，点「启动相机」。
6. 观察：相机是否打开，骨架是否贴合人体，性能面板是否实时变化。
7. 依次点「切换 delegate」，记录 CPU/NNAPI/GPU 是否可用、FPS 是否变化。
8. 如骨架方向不对，试「旋转90」「镜像」，记录哪种组合正确。

```text
App 显示的错误文本：
```

## 输出文件

App 会写出：

- `app_keypoints.csv`
- `performance_report.json`
- `live_performance_report.json`

这些文件优先在 Android app-specific external 目录，拿不到外部目录时回退到 App documents 目录。
