## 3. 计数算法（本阶段成果，已验证）

**文件**：`lib/pushup_domain.dart` 的 `PushupCounter` 类。

**算法**（armed/reset 峰谷检测，纯信号形状，帧率无关）：
```
shoulder_y → 中值滤波(window=5) → 5%/95% 分位数算鲁棒幅值 amp
amp < 80px：不判定（覆盖静止 + 冷启动）
否则 THR = max(0.5·amp, 80px)
  armed：越过 enterDown(p05+0.65·amp) 且 (y − 自上次计数最低点) ≥ THR → 计数+1，disarm
  disarm：回落到 enterUp(p05+0.35·amp) → 重新 arm，重置低点跟踪
```

**关键参数（`CounterConfig`）**：`ampMinPx=80`、`thrRatio=0.5`、`hystHigh=0.65`、`hystLow=0.35`、`pLow=0.05`、`pHigh=0.95`、`confThr=0.3`。旧的状态机字段（windowN/minGapFrames 等）保留为 `@Deprecated` 兼容 App 调用，**未被使用**。
